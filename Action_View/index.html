Action View

Chú ý: Những phần nào còn thiếu thì tự tạo thêm.

  • Cách tạo:

1_ Tạo file custom/modules/<MODULE_NAME>/views/view.<VIEW_FACTORY_NAME>.php với nội dung:

<?php
class MODULE_NAMEViewACTION_NAME extends SugarView
{
    function MODULE_NAMViewACTION_NAME()
    {
        parent::SugarView();
    }

    public function preDisplay()
    {
        parent::preDisplay();
    }

    public function display()
    {
        parent::display();
    }
}

2_ Tạo file custom/modules/<MODULE_NAME>/action_view_map.php với nội dung:

<?php
 $action_view_map['<ACTION_NAME>'] = '<VIEW_FACTORY_NAME>';

3_ Truy cập vào <PROJECT_URL>/index.php?module=<MODULE_NAME>&action=<ACTION_NAME> để xem action.

Lưu ý:

- VIEW_FACTORY_NAME: Viết thường toàn bộ.
- MODULE_NAME: thay bằng tên module muốn để action.
- ACTION_NAME: viết theo kiểu lạc đà.
- Nếu muốn dùng action view để xử lý Ajax thì thêm dòng &sugar_body_only=true vào cuối url
