Thêm css và js vào các màn hình EditView, DetailView

Chú ý: Những phần nào thiếu thì tự tạo thêm.
  • Vị trí:
    • custom/Module/<module>/metadata/
    • Khai báo trong array templateMeta
  • Cú pháp:
'javascript' => '
    //Thêm js
   {sugar_getscript file="<file_location>"}
   //Thêm css
    <link rel=\'stylesheet\' href=\'{sugar_getjspath file="<file_location>"}\'/>
',
  • Ví dụ thêm css và js vào màn hình EditView module J_Class:
 'javascript' => '
       {sugar_getscript file="custom/modules/J_Class/js/editviewt.js"}
       <link rel=\'stylesheet\' href=\'{sugar_getjspath file="custom/modules/J_Class/css/style_nd.css"}\'/>
  ',
