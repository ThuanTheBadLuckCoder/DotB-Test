Tạo mới Fields bằng vardefs

Chú ý: Những phần nào thiếu thì tự tạo thêm.
  • Vị trí: custom/Extension/Module/<module>/Ext/vardefs/vardef.php
  • Các loại type thường dùng:
Type Properties bắt buộc Ý nghĩa
varchar, text, date, datetime, int, float, decimal Không có Do những kiểu dữ liệu này trùng với trong DB
radioenum 'dbType' => 'varchar', Do kiểu dữ liệu này không có sẵn trong DB nên phải khai báo kiểu dữ liệu của nó trong DB
float, decimal, currency 'len' => 8, 'precision' => 2 Để khai báo số chữ số và số chữ số thập phân
enum, multienum, radioenum 'options' => 'account_gender_dom', Những field này cần khai báo thêm danh sách các option để chon lựa
  • Các properties đặc biệt:
    • audit: dùng để khai báo theo dõi lịch sữ field
    • import: field được khai báo có thể dùng để import record mới vào module
    • search: field được khai báo có thể dùng để tìm kiếm record của module
    • len: số kí tự (đối với field chữ), số chữ số (đối với field số)
    • required: bắt buộc nhập hay không.

  • Ví dụ 1

    • Tạo fields varchar:

    $dictionary['Account']['fields']['first_name'] = array( 'name' => 'first_name', 'type' => 'varchar', 'label' => 'LBL_FIRST_NAME', 'requited' => true, );

  • Ví dụ 2

    • Tạo fields enum:

    $dictionary['Account']['fields']['gender'] = array( 'name' => 'gender', 'type' => 'enum', 'label' => 'LBL_LABEL', 'options' => 'account_gender_dom', );

  • Ví dụ 3

    • Tạo fields int:

    $dictionary['Account']['fields']['age'] = array( 'name' => 'age' , 'type' => 'int', 'label' => 'LBL_AGE', 'len' => 3, );

  • Để thêm Label cho Field vào File custom/modules/<MODULE_NAME>/language/<LANGUAGE>.lang.php (nếu không có thì tạo thêm). Thêm vào nội dung sau:

<?php
    $mod_strings['<LABEL_NAME>'] = '<TRANSLATE>';
    // Ví dụ

    $mod_strings['LBL_FIRST_NAME'] = 'First name';
  • Để thêm option cho Field vào File custom/include/language/<LANGUAGE>.lang.php (nếu không có thì tạo thêm). Thêm vào nội dung sau:
<?php
    $app_list_strings['<DOM_NAME>'] = array(
        '<KEY1>' => '<VALUE1>',
        '<KEY2>' => '<VALUE2>',
    );
    // Ví dụ
    $app_list_strings['account_gender_dom'] = array(
            'Male' => 'Male',
            'Female' => 'Female',
            'Other' => 'Other',
     );
