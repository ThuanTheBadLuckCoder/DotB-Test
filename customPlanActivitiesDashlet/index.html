Custom filter Dashlet Plan Activities

Mở file clients/base/views/planned-activities/planned-activities.js và tìm đến hàm _getFilters. Đây là hàm xử lý filter của dashlet này. Tại đây ta tiến hành custom lại giá trị của biến filter như sau:

ví dụ: filter các meeting có location là HCM ở dashlet của module Leads

_getFilters: function (index) {

        var tab = this.tabs[index];
        var filter = {};
        var filters = [];

        if (this.module == 'Leads' && tab.module == 'Meetings') {
            filter['location'] = {$equals: 'hcm'};
        }

        filters.push(filter);

        return filters;
}

Các operation filter

$equals, $not_equals, $starts, $ends, $contains, $not_contains, $in, $not_in, $dateBetween, $between, $is_null, $not_null, $empty, $not_empty, $lt, $lte, $gt, $gte, $dateRange

Lưu ý: $in requires an array, $not_in requires an array, $between requires an array with two values.
