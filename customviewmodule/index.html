  • Create a custom view module

ví dụ tạo màn hình customize ở module Leads với tên là leadtest

Tạo layout

Tạo thư mục và 2 file trong đó gồm 1 file js và 1 file php có cùng tên với nhau theo đường dẫn custom/modules/Leads/clients/base/layouts/leadtest

  • leadtest.php

Khai báo layout và view mà layout sẽ sử dụng để render.

<?php
$viewdefs['Leads']['base']['layout']['leadtest'] = array(
    'components' => array(
        array(
            'layout' => array(
                'type' => 'base',
                'name' => 'main-pane',
                'css_class'=>'main-pane',
                'components' => array(
                    array(
                        'view' => 'leadtest',
                    ),
                )
            )
        )
    )
);
  • leadtest.js init layout và load data
({
    initialize: function (options) {
        this._super('initialize', [options]);
    },

    loadData: function (options) {
        var data = {
            fielda: 'abc',
            fieldb: 'def'
        };
        this.model.set(data, {silent: true});
    }
})

Tạo view

Tạo thư mục và 3 file khai báo view cùng tên leadtest theo đường dẫn sau: custom/modules/Leads/clients/base/views/leadtest

  • leadtest.php Khai báo các button của view
<?php
$viewdefs['Leads']['base']['view']['leadtest'] = array(
    'buttons' => array(
        array(
            'name' => 'cancel_button',
            'type' => 'button',
            'label' => 'LBL_CANCEL',
            'css_class' => 'btn-invisible btn-link',
            'events' => ["click" => "cancel"]
        ),
        array(
            'type' => 'button',
            'name' => 'save_button',
            'label' => 'LBL_SAVE',
            'css_class' => 'btn-primary',
            'events' => ["click" => "save"]
        )
    )
);
  • leadtest.js khai báo các hàm xử lý
({
    initialize: function (options) {
        this._super('initialize', [options]);

        this.context.on("save", _.bind(this.save, this));
        this.context.on("cancel", _.bind(this.cancel, this));
    },

    save: function () {
        alert('click save');
    },

    cancel: function () {
        alert('click cancel')
    }
})
