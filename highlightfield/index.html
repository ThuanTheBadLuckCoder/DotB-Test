Định nghĩa field là field event-status

*Lưu ý: chỉ có field type là enum

Trong phần khai báo danh sách các field của các màn hình listview, record hay subpanel thêm type cho field là 'type' => 'event-status'. Nếu muốn highlight full-width thì thêm khai báo 'css_class' => 'full-width'.

ví dụ: 

array(
    'name' => 'status_trial',
    'label' => 'LBL_STATUS_TRIAL',
    'type' => 'event-status',
    'css_class' => 'full-width',
)

Định nghĩa màu highlight

Ta định nghĩa màu highlight trong file clients/base/fields/event-status/event-status.js

Ở đây ta sẽ định nghĩa giá trị nào của field thì hiển thị với class label nào.

this.statusClasses = {
    'Held': 'label-success',
    'Not Held': 'label-important',
    'Planned': 'label-pending'
};
