Hệ thống được khuyến khích chạy trên nền tảng với Apache 2.4, PHP 7.1, MySQL 5.7, ElasticSearch 6.7

PHP Requirements

Cài các extionsion sau cho php: bcmath,curl, gd,gmp,hash,imap,json,mbstring,mcrypt,openssl,SimpleXML,zip,zlib

Cài 1 trong các extionsion sau để provide cho function random token được sử dụng trong hệ thống: libsodium,mcrypt ,com_dotnet

Thiết lập các giá trị mặc định cho php:

date.timezone=Asia/Ho_Chi_Minh
display_errors=0
fastcgi.logging=0
max_execution_time=3600
max_input_time=3600
mbstring.func_overload=0
memory_limit=2048M
post_max_size=1024M
session.use_cookies=1
session.cookie_httponly=1
session.use_trans_sid=1
upload_max_filesize=1024M
max_input_vars = 1000000

Lưu ý: Nếu sử dụng WAMP thì phải tắt extension opcache.

Database Requirements

Thiết lập các thông số sau trong my.ini

default-storage-engine=InnoDB
sql-mode="STRICT_ALL_TABLES,ERROR_FOR_DIVISION_BY_ZERO,NO_ZERO_DATE,NO_ZERO_IN_DATE,NO_AUTO_CREATE_USER"

Config Apache server

Mở file httpd.conf và uncomment các dòng để cho phép chạy các module sau: mod_rewrite , mod_headers , mod_expires, mod_deflate

Install Elasticsearch

sudo yum install java-1.8.0-openjdk.x86_64
rpm --import https://artifacts.elastic.co/GPG-KEY-elasticsearch

Create a file called elasticsearch.repo in the /etc/yum.repos.d/ directory

vi /etc/yum.repos.d/elasticsearch.repo

press i and past file content

[elasticsearch-6.x]
name=Elasticsearch repository for 6.x packages
baseurl=https://artifacts.elastic.co/packages/6.x/yum
gpgcheck=1
gpgkey=https://artifacts.elastic.co/GPG-KEY-elasticsearch
enabled=1
autorefresh=1
type=rpm-md

And your repository is ready for use. You can now install Elasticsearch with one of the following commands:

sudo yum install elasticsearch
sudo /bin/systemctl daemon-reload
sudo /bin/systemctl enable elasticsearch.service
sudo systemctl start elasticsearch.service

Test: curl -X GET http://localhost:9200

Config ElasticSearch

Link download ES: https://www.elastic.co/downloads/elasticsearch

Mở file config.php trong source code

$sugar_config['full_text_engine']['Elastic']['host'] = 'localhost';
$sugar_config['full_text_engine']['Elastic']['port'] = '9200';
