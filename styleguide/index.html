Grid system

Giao diện được chia làm 12 cột trên 1 hàng ngang.

  • general
<div class="row-fluid">
  <div class="span4">...</div>
  <div class="span8">...</div>
</div>
  • offset
<div class="row-fluid">
  <div class="span4">...</div>
  <div class="span4 offset4">...</div>
</div>
  • child grid of a column 
<div class="row-fluid">
  <div class="span6">
    Level 1 of column 1
    <div class="row-fluid">
        <div class="span6">Level 2</div>
        <div class="span6">Level 2</div>
    </div>
  </div>
  <div class="span6">
    Level 1 of column 2
  </div>
</div>
  • Ngoài ra, có thể sử dụng thẻ div với class .container-fluid để padding.
<div class="container-fluid">
  <div class="row-fluid">
    <div class="span8">
      <!--Body content-->
    </div>
    <div class="span4">
      <!--Sidebar content-->
    </div>
  </div>
</div>

Button

Có các loại button sau: - btn - btn-primary - btn-info - btn-success - btn-warning - btn-danger - btn-inverse - btn-link

Cách sử dụng: chèn các class trên vào các thẻ a,button,input mà bạn muốn apply css của button đó vào. Lưu ý class btn là bắt buộc đi kèm.

<a class="btn" href="">Link</a>
<button class="btn btn-success" type="submit">Button</button>
<input class="btn btn-primary" type="button" value="Input">
<input class="btn btn-danger" type="submit" value="Submit">

Button size: btn-large, btn-small, btn-mini

Button fullwidth: .btn-block

Button disabled: .disabled

Invisible button background: .btn-invisible

Button groupd: 

<div class="btn-group">
  <a class="btn">1</a>
  <a class="btn">2</a>
  <a class="btn">3</a>
</div>

<div class="btn-toolbar">
  <div class="btn-group">
    ...
  </div>
</div>

Button dropdown:

<div class="btn-group">
  <a class="btn dropdown-toggle" data-toggle="dropdown">
    Action
    <i class="fa fa-caret-down"></i>
  </a>
  <ul class="dropdown-menu">
    <!-- dropdown menu links -->
  </ul>
</div>

Split button dropdowns:

<div class="btn-group">
  <a class="btn">Action</a>
  <a class="btn dropdown-toggle" data-toggle="dropdown">
    <i class="fa fa-caret-down"></i>
  </a>
  <ul class="dropdown-menu">
    <!-- dropdown menu links -->
  </ul>
</div>
